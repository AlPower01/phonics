Project Goal
Transform phonics.org from a flat blog structure into a hierarchical, topic-clustered authority site that establishes topical depth, serves dual audiences (parents + teachers), and generates qualified leads for reading.com.
Success Metrics
  • All content reachable in ≤ 3 clicks
  • Average 5-8 internal links per page
  • 30% increase in organic traffic
  • Improved AI citation/ranking (GEO)
How to Use These Wireframes
  1. Review each wireframe by clicking the cards below
  2. Read annotations (marked with 📐) for design specifications
  3. Note the component reuse — many elements repeat across pages
  4. Check responsive notes — all layouts adapt to mobile
  5. Reference the PDR document (phonics-org-design-brief.md) for complete specifications
All Wireframes
1
Homepage
Authority gateway routing users to silos. Includes hero, silo cards, featured content, reviews spotlight, age pathways, latest articles, trust bar, and newsletter CTA.
URL: /
2
Mega Menu Navigation
The most important architectural change. 4-column dropdown panels exposing full silo structure. Includes desktop hover and mobile accordion versions.
7 unique menus
3
Pillar / Hub Page
Comprehensive 3,000-5,000 word overview linking to all cluster pages in the silo. Includes TOC sidebar, cluster CTA cards, FAQ accordion, and cross-silo links.
Template for 7 silos
4
Article / Cluster Page
Individual articles within silos. Two-column layout (65/35) with sticky sidebar, key answer box, audience callouts, related articles, and next steps navigation.
URL: /[silo]/[topic]/[article]/
5
Reviews Hub
Filterable directory of 58+ app reviews. Includes editor's picks, sticky filter bar (age, platform, price, rating), review grid, pagination, and comparison tables.
URL: /reviews/
6
Single Review Page
Standardized app review template. Prominent score badge, visual score breakdown, quick verdict, pros/cons, detailed review body, and alternatives section.
URL: /reviews/[app-name]/
7
Audience Landing Page
Curated entry points for parents/teachers (NOT archives). Age-based reading paths with progressive disclosure, popular resources grid, and recommended apps.
URLs: /for-parents/, /for-teachers/
8
Glossary
Two-part system: Hub with alphabetical index of 50-80 terms, plus individual term pages (300-500 words each) with definitions, explanations, and related links.
URLs: /glossary/, /glossary/[term]/
