Mega Menu System Overview
Purpose: The mega menu is the primary navigation mechanism. It must expose the full silo structure in a scannable, visually organized panel. Each of the 7 nav items opens a mega menu with this 4-column structure.
Specifications
Trigger Hover on desktop (200ms delay), tap on mobile
Panel Width 100% viewport width, max 1200px centered
Panel Height Auto-fit content, max ~400px with scroll
Animation Fade + slide down, 150ms ease-out
Mobile Full-screen accordion panels within hamburger drawer

Implementation Notes for Developer
Key Requirements
  • All 7 nav items need unique mega menu content (examples shown: Skills, Parents)
  • Hover delay: 200ms before opening (prevents accidental triggers)
  • Close behavior: Click outside, ESC key, or click another nav item
  • Semi-transparent backdrop when mega menu is open
  • Accessibility: Keyboard navigation (Tab, Enter, ESC), ARIA labels
  • Mobile hamburger: Positioned top-right in header
  • Featured content: Should pull latest from each silo (dynamic)
